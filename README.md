
# newton-website-2.0

1. **Main technology stack**

    ```shell
    Gatsby.js + React + Typescript + Tailwind CSS.
    Other tools included: Autoprefixer, Sass, PostCSS, ESLint, Prettier.
    ```

1.  **Create a Gatsby site.**

    ```shell
    gatsby new newton-website-2.0
    yarn
    ```

2.  **Start**

    ```shell
    cd newton-website-2.0/
    yarn start
    ```

3. **push**
    ```shell
    yarn build
    git push
    ```
